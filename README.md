# quiz-app-backend

A simple backend for my quiz app.

## Installation
`npm install` or `yarn install`

## Execution
For autoreload, run `npm run dev` or `yarn dev`.
For regular execution, run `npm start` or `yarn start`

## Testing
`npm run test` or `yarn test`

## Stack
- Node.js
- Express
- Nodemon
- Colorful-log
- Mongoose
